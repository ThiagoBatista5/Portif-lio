# Portfolio Pessoal - Thiago Batista

## 📱 Introdução

Este é o **portfolio pessoal** de **Thiago Batista**, Desenvolvedor Frontend. Um site one-page moderno, responsivo e animado, construído com HTML5, CSS3 e JavaScript puro.

**Foco**: Apresentar habilidades, projetos e contato de forma elegante e performática, com design dark theme inspirado em tendências atuais (glow effects, smooth scrolls, reveals).

## ✨ Funcionalidades

- **Navegação sticky**: Navbar com scroll effect (backdrop blur).
- **Animações suaves**: Reveal on scroll com IntersectionObserver (staggered).
- **Seções completas**:
  - Hero com CTA (projetos, WhatsApp).
  - Sobre (bio, stats: 1+ ano exp., 10+ projetos).
  - Habilidades (chips: React, Next.js, Tailwind, Node.js, Figma, etc.).
  - Projetos (cards com previews mockados: ShopFlow, Sistema Financeiro, Gerenciador Tarefas).
  - Contato (Email, LinkedIn, GitHub) + Footer.
- **Responsivo**: Mobile-first (768px breakpoint).
- **Performance**: CSS vars, minimal JS, noise overlay SVG.
- **Acessibilidade**: Semântica HTML, hover states.

## 🛠️ Tecnologias

| Frontend | Estilo | JS | Outros |
|----------|--------|----|--------|
| HTML5 | CSS3 (custom properties, grid/flex) | Vanilla JS (ES6+) | Google Fonts (Syne, DM Sans) |

- **Animações**: IntersectionObserver, CSS @keyframes (fadeUp, pulse, etc.).
- **Efeitos**: Glows, backdrop-filter, scroll-behavior smooth.
- **Icons**: SVGs inline.

## 🚀 Instalação e Uso

Nenhuma instalação necessária (estático)!

1. Abra `index.html` no navegador:
   ```bash
   open index.html  # macOS
   # ou use Live Server no VSCode
   ```

2. Navegue pelas âncoras (#about, #skills, etc.).
